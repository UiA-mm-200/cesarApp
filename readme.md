## Detaljer
Bruker:
- Node
- Express.js
- Body-parser.js
- pg.js for å prate med en Postgres SQL database (installers som "adon" på heroku)
- Html for client.

Vi hoster appen i europa og vi bruker en gratis versjon (hobby) av Heroku Postgres
